# פסיכומטרי — חלק כמותי

אפליקציית ווב לתרגול החלק הכמותי של הפסיכומטרי.

## מבנה הפרויקט

```
psychometric/
├── index.html              # דף הבית
├── style.css               # עיצוב משותף
├── shared/
│   ├── engine.js           # לוגיקת סשן, טיימר, ניקוד, ניתוח תשובות
│   └── ui.js               # רכיבי תצוגה משותפים
├── drill/
│   ├── index.html          # מסך תרגול
│   └── drill.js            # יצירת תרגילי חשבון
└── simulator/
    ├── index.html          # מסך סימולטור
    └── questions.js        # יצירת שאלות מילוליות
```

## העלאה ל-GitHub Pages

1. ליצור repo חדש בגיטהאב (לדוגמה `psychometric`)
2. להעלות את כל התיקייה תוך שמירה על מבנה התיקיות (כולל `shared/`, `drill/`, `simulator/`)
3. ב-Settings → Pages, לבחור Source: `main` branch, root
4. להמתין דקה ולגלוש לכתובת `https://<username>.github.io/<repo-name>/`

## הוספה למסך הבית באייפון

1. לפתוח את הקישור ב-Safari
2. ללחוץ על כפתור Share (✦) → "Add to Home Screen"
3. האייקון יופיע במסך הבית כמו אפליקציה רגילה

## נושאים מכוסים

**תרגול:** חיבור, חיסור, כפל, חילוק, חזקות, שורשים, ביטויים מעורבים.

**סימולטור:**
- חזקות (השוואת בסיסים, גורם משותף)
- כפל מקוצר והפרש ריבועים
- מערכת משוואות, ערך מוחלט
- פעולות מומצאות
- יחס משולב, אחוזים עוקבים
- ממוצע, סדרה חשבונית, סדרה הנדסית
- שאריות בחילוק

כל השאלות נוצרות פרוצדורלית בכל סשן — אין מאגר קבוע.

## הערות פיתוח

- Vanilla JS בלבד, ללא build step
- מותאם לאייפון (RTL, safe-area, touch targets 44px+)
- תומך במצב כהה אוטומטי (לפי הגדרות המכשיר)
- בכתיבה חופשית ניתן להזין שברים בפורמט `3/4` או עשרוני `0.75`
